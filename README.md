# Messy Data Cleaning Project

## Project Overview

This project focuses on cleaning real-world messy sales data using both Excel and Python.

## Objectives

* Handle missing values
* Remove duplicate records
* Standardize inconsistent data (text & dates)
* Perform basic data analysis

## Tools Used

* Microsoft Excel
* Python (Pandas)

## Cleaning Steps

* Removed duplicate rows
* Filled missing values using logical defaults
* Standardized text fields (Plan column)
* Converted inconsistent date formats into a uniform format

## Files Included

* messy_sales_data.csv (raw dataset)
* clean_sales_excel.csv (cleaned using Excel)
* clean_sales_data.csv (cleaned using Python)
* Python scripts for automation

## Key Insights

* Sales distribution by country
* Sales performance by plan
* Identification of top customers

## Conclusion

This project demonstrates practical data cleaning skills used in real-world scenarios and highlights the importance of preparing data before analysis.
